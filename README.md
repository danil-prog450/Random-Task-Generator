Имя: [Данила]

## Описание

Приложение "Random Task Generator" генерирует случайные задачи из предопределенного списка.  Пользователь может генерировать задачи, добавлять новые задачи, фильтровать историю сгенерированных задач по типу и сохранять историю в файл.
