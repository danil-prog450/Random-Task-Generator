# Book Tracker

## Описание

Book Tracker - это графическое приложение на Python, разработанное с использованием библиотеки Tkinter, для управления списком прочитанных книг. Приложение позволяет добавлять новые книги, фильтровать их по жанру и количеству страниц, а также сохранять и загружать данные из JSON-файла.

## Функциональность

*   **Добавление книг:** Пользователь может ввести название, автора, жанр и количество страниц книги, а затем добавить ее в список.
*   **Фильтрация:**
    *   По жанру (частичное совпадение).
    *   По количеству страниц (больше указанного значения).
*   **Сохранение и загрузка данных:** Данные о книгах сохраняются в формате JSON и загружаются при запуске приложения.
*   **Валидация ввода:** Проверка на заполнение всех полей и корректность ввода количества страниц (положительное целое число).

## Инструкция по запуску

1.  **Установка Python:** Убедитесь, что у вас установлен Python (версия 3.7 или выше).
2.  **Установка Tkinter:** Tkinter обычно входит в стандартную установку Python. Если у вас его нет, переустановите Python.
3.  **Клонирование репозитория:**
4.  **Запуск приложения:**
    python main.py
5.  **Запуск тестов:**
    test_book_tracker.py
 

## Примеры использования

1.  **Добавление книги:**
    *   Введите данные о книге в соответствующие поля (Название, Автор, Жанр, Количество страниц).
    *   Нажмите кнопку "Add Book".
    *   Появится сообщение об успешном добавлении.
2.  **Фильтрация по жанру:**
    *   Введите жанр в поле "Genre".
    *   Нажмите кнопку "Filter".
    *   В списке отобразятся книги, соответствующие введенному жанру.
3.  **Фильтрация по количеству страниц:**
    *   Введите количество страниц в поле "Pages".
    *   Нажмите кнопку "Filter".
    *   В списке отобразятся книги, у которых количество страниц больше указанного.
4.  **Сочетание фильтров:**
    *   Введите жанр и количество страниц.
    *   Нажмите кнопку "Filter".
    *   В списке отобразятся книги, соответствующие обоим критериям.
